## Datos

Los datos se obtienen de la Chess.com Public API:

```
https://api.chess.com/pub/player/{username}/games/{year}/{month}
https://api.chess.com/pub/player/{username}/stats
```

No requiere autenticación para datos de perfiles públicos.
